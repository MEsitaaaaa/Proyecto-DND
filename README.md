# Proyecto-DND
Este proyecto implementa una jerarquía de clases en Java para un sistema de personajes de videojuego, siguiendo los principios de la Programación Orientada a Objetos.

## Requisitos del Examen
El desarrollo cumple con los siguientes puntos solicitados en la recuperación de la UD4:
Clase Padre Personaje: Incluye atributos de nombre, nivel y energía, además de lógica para entrenamiento y gestión de recursos.
Clases Hijas: Implementación de las clases Guerrero (con ataque pesado) y Mago (con ataque elemental).
Miembros Estáticos: Control de un contador global de personajes creados mediante atributos y métodos estáticos.
Clase Ejecutable: Clase App que realiza las pruebas de instanciación, uso de métodos y gestión de arrays.
## Flujo de Trabajo 
Siguiendo la estructura de ramas solicitada en clase:
main: Rama principal para la entrega final.
feature: Rama integradora de todas las funcionalidades.
Ramas específicas: Cree ramas independientes para cada componente (personaje, guerrero, mago y App) para asegurar un desarrollo organizado y secuencial
